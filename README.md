# ToDo List

A learning project built with vanilla JavaScript to practice basic task management, data handling, and user interactions.

## Project Purpose
This project is created to practice:
- Managing list-based data (tasks)
- Implementing basic CRUD operations
- Persisting data using browser LocalStorage
- Handling user interactions with a clean UI flow

## Features
- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Store tasks in LocalStorage so data remains after page refresh

## Technologies Used
- JavaScript (Vanilla JS)
- HTML
- CSS
- Browser LocalStorage
- Git / GitHub

## How to Run This Project
- Open `index.html` directly in a browser  
- Or use VS Code Live Server for better development experience

## Learning Outcomes
- Understanding CRUD concepts through task list management
- Practicing DOM manipulation and event handling
- Managing client-side data persistence with LocalStorage
- Structuring a small frontend project clearly

## Notes
This project is created for learning and practice purposes and is not intended for commercial use.
